How Do I Convert Certificates With OpenSSL?

Create a rsa key:

openssl genrsa -out sample.key 4096

Convert .crt (and private key) to .p12:

openssl pkcs12 -export -in sample.crt -inkey sample.key -out sample.p12

Convert .pem to .der:

openssl x509 -outform der -in sample.pem -out sample.der

Convert .der to .pem:

openssl x509 -inform der -in sample.der -out sample.pem

Convert .p12 to .key:

openssl pkcs12 -in sample.p12 -nocerts -out sample.key

Convert .p12 to .crt:

openssl pkcs12 -in sample.p12 -clcerts -nokeys -out sample.crt
This site is not intended to be an exhaustive list of all possible commands.
If you find yourself needing commands not listed here, simply create a pull request at GitHub.
